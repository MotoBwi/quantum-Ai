# Quantum AI-react-sample

This repository contains React samples and recipes that demonstrate use cases using the Quantum AI platform.

### Quantum AI Platform
- Website: [https://www.Quantum AI.ai/](https://www.Quantum AI.ai/)
- Documentation: [https://doc.Quantum AI.ai/introduction/overview](https://doc.Quantum AI.ai/introduction/overview)

### NPM Package
To get started, install the official Quantum AI React package:
```bash
npm install @Quantum AIai/react
```

Explore examples in this repository to learn how to integrate and use Quantum AI efficiently in different scenarios.

## Use Cases and Examples

| **Use Case**         | **File**                                    | **Description**                                          |
|----------------------|---------------------------------------------|----------------------------------------------------------|
| Web voice agent      | `src/examples/web-app.tsx`                  | Embed a voice + text AI agent in a web app               |
| Make a phone call    | `src/examples/make-phone-call.tsx`          | Initiate an outbound phone call via the Quantum AI API       |

## Getting Started

Install dependencies:

```bash
npm install
```

Start the dev server:

```bash
npm start
```
